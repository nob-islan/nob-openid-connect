# エンドポイント一覧

各種画面 URL および API のエンドポイントを定義します。

## 認証開始用 URL

`/start`

### 画面 URL

| 画面名                   | URL          |
| ------------------------ | ------------ |
| 認証画面                 | `/login`     |
| ユーザ情報提供の確認画面 | `/agreement` |
| トップ画面               | `/top`       |

## リライングパーティ API エンドポイント

| API 名                         | URL                   |
| ------------------------------ | --------------------- |
| 認可エンドポイントリダイレクト | `/api/redirect/login` |
| トークンリクエスト             | `/api/token/request`  |
| ID トークン検証                | `/api/token/verify`   |
| ユーザプロフィール要求         | `/api/user/profile`   |
| トップ画面リダイレクト         | `/api/redirect/top`   |

## ID プロバイダ API エンドポイント

| API 名             | URL                  |
| ------------------ | -------------------- |
| 認可エンドポイント | `/api/authorization` |
